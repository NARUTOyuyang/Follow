### APP类型：
>
这是一个把平时在我们项目里常用的功能和 UI 效果展示出来的 APP。

### 常用的效果有：
>
* 登入（缓存），加载页面时缓存动画；退出。

* 常用组件的封装引用（Modal）。
* HeadBar` 和 TabBar。

* 视频播放和录制。
* 导航定位。
